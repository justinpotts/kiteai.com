Installation

To call a Rest API in your project, you'll need to install a library that can make GET and POST requests. For this guide, we'll use unirest, a popular library for Rest APIs because of its support for multiple languages. To install unirest for Python, it's as easy as using pip.

$ pip install unirest

Visit http://unirest.io/ on how to install this for other languages.

If you're not sure which language you're going to use yet, we recommend Postman, a downloadable app that makes it easy to make calls to APIs. It's great for testing your keys and experimenting with different endpoints.

Additionally, if you're familiar with the command line, you can use curl.

Creating an API key

1. Create an account on https://developer.kiteai.com, and create a new app. We'll call this app "My first app." You can leave the description blank.

2. After creating My First App, click on the app in the Apps Dashboard to enter its details, and click the API Key field to reveal the key.

Your first call

After setting up your environment and creating your first app, you're ready to make your fisrt call! We'll use the three methods above to show you how to make a call.

Python using unirest:

$ python
>>> import unirest
>>> response = unirest.get("https://api.kiteai.com/predict?phrase=Testing out my first app!",
headers={
"x-api-key": "my-first-app-key",
"Accept": "application/json"
})
>>> print(response.body)

To use Postman, change the method type to POST and add a header called x-api-key with your API key as the value.

Using curl

$ curl --get --include 'https://api.kiteai.com/predict?phrase=Testing out my first app!' -H 'x-api-key: my-first-app-key' -H 'Accept: application/json'
